About Nimbus
----------------------

Nimbus is a open source Framework with below features.

Customized Workflow - Users can configure the process flow depends on their busseness needs.
Configurable UI - No need to write the code to develop the UI ..just need to use annotations to get the UI component.
Multi Tenancy - Supports multiple tanant (Saas)
Cloud based solution
Customized and configurable rules.
Providing the ability to build application through configuration.
Providing boilerplate code for cross cutting concerns.

spring data
junit

YAML - For confguration files
Activiti designer - Activiti is a light-weight workflow and Business Process Management (BPM) Platform targeted at business people, developers and system admins
